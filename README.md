# 🤖 Wibes Admin Bot — Инструкция

## Что умеет бот

| Команда | Что делает |
|---------|-----------|
| `/gencode` | Создать 1 код доступа |
| `/gencode 10` | Создать 10 кодов сразу |
| `/codes` | Показать все неиспользованные коды |
| `/used` | Показать кто уже вошёл |
| `/withdrawals` | Все запросы на вывод (ник, адрес, сумма) |
| `/stats` | Общая статистика |

---

## Установка (5 минут)

### Вариант А — Replit (бесплатно, просто)

1. Зайдите на **replit.com**, создайте аккаунт
2. Нажмите **Create Repl** → выберите **Python**
3. Загрузите файлы `bot.py` и `requirements.txt`
4. В разделе **Secrets** (замок слева) добавьте:
   - `BOT_TOKEN` = ваш токен от @BotFather
   - `ADMIN_ID` = ваш Telegram ID (узнайте у @userinfobot)
5. Нажмите **Run** ✅

### Вариант Б — Railway (надёжнее, тоже бесплатно)

1. Зайдите на **railway.app**, войдите через GitHub
2. **New Project** → **Deploy from GitHub repo**
3. Загрузите файлы на GitHub, подключите репо
4. В **Variables** добавьте `BOT_TOKEN` и `ADMIN_ID`
5. Railway запустит бота автоматически ✅

## Получить BOT_TOKEN

1. Напишите **@BotFather** в Telegram
2. `/newbot` → придумайте имя → придумайте username (должен заканчиваться на `bot`)
3. Получите токен вида: `7123456789:AAHdqTcvCHhvQNKMe1bEcScvnYLGOPfeDGk`

## Получить ADMIN_ID

Напишите боту **@userinfobot** — он ответит вашим ID, например: `123456789`

## Интеграция сайта с ботом

В файле `index.html` найдите строки 293-296 и заполните:

```javascript
const BOT_TOKEN = '7123456789:AAHdqTcvCHhvQNKMe1bEcScvnYLGOPfeDGk';
const CHAT_ID   = '123456789';  // ваш Telegram ID
```

Для `VALID_CODES` — оставьте пустой массив, коды будут проверяться динамически:

```javascript
// Способ 1: статические коды (простой)
const VALID_CODES = []; // коды генерирует бот, сайт проверяет через API

// Способ 2: вручную копируете код из бота и вставляете сюда
const VALID_CODES = ['ABCD-1234', 'EFGH-5678'];
```

> **Совет:** Самый простой способ — после `/gencode` скопировать код,  
> вставить в `VALID_CODES` в `index.html` и заново загрузить на Netlify.  
> Занимает 30 секунд.
